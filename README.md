# Comp-2007-Coursework


Treasure Street Escape is a first-person exploration game where the player must navigate a small street environment and collect hidden treasures before time runs out.

Features:
- First-person controls (WASD + mouse)
- Randomised treasure locations
- Timer-based gameplay
- Interactive pickup system (E key)
- Win and lose conditions
- UI menus and feedback system

Controls:
WASD – Move
Mouse – Look
Space – Jump
E – Collect treasure

Installation:
Download and extract the ZIP file, then run the .exe.
